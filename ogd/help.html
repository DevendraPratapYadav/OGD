Loading datasets

To load a dataset for prediction, do the following,

  1. Enter the name of the dataset you want to run your prediction on and press 'Load'.
  2. Upon doing that you can view the data and its visualisation right underneath the search bar.
  3. In the data shown, change pages to view more of the dataset.
  4. In the visualisation, choose from various options like
    • Column Chart
    • Area Chart
    • Line Chart
    to view different visualisations of the data.

Viewing correlations

After the data has been loaded, to view the correlations among various columns of the data,

  1. From the given list of columns, select those whose correlations you would like to see and press 'Continue Selection'
  2. Upon doing that an image appears that shows the correlations among the different columns selected.

Making predictions

To make predictions on a dataset after the aforementioned steps are complete, do the following

  1. For every column select a radio button out of 'Input' , 'Output' and 'None'. This tells the program whether the column has to be used as input, output or not used at all.
  2. A prediction model is trined using all the columns marked as input as database, to produce an output for all the columns marked as output
  3. To select how many values have to be predicted, enter the following
    • Start - from where the value starts
    • Increment - steps in which the numbers for which predictions have to be made are chosen
    • number of values - For how many values a prediction has to be made
  4. For example for the given values, Start - 2010, Number of values - 10, Increment - 1, predictions are made for all the years from 2010-2019 ( both inclusive )
  5. Upon doing that, an image appears that shows the data predicted by the code overlayed on the actual dataset.
  6. The presictions also appear as a table beneath the image
